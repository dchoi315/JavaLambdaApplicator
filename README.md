# JavaLambda

##Installation

Clone the repository and run the following command in thedirectory with the lambda.jar file

'''
java -jar lambda.jar
'''

A new terminal will open up where we can begin running commands

##Usage

####Exiting the program
'''
> exit
'''

####Creating a Variable
A variable is any set of characters
'''
> a
a
> b
b 
> T
T
> Github
GitHub
'''
